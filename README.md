## required libraries to get this all working
1. [arduino-1.6.8/hardware/esp8266com/esp8266/libraries/SPI/SPI.h](https://github.com/esp8266/Arduino/blob/master/libraries/SPI/SPI.h)
2. [arduino-1.6.8/hardware/esp8266com/esp8266/libraries/Wire/Wire.h](https://github.com/esp8266/Arduino/blob/master/libraries/Wire/Wire.h)
3. [Arduino/libraries/Adafruit_GFX_Library/Adafruit_GFX.h](https://github.com/adafruit/Adafruit-GFX-Library/blob/master/Adafruit_GFX.h)
4. [Arduino/libraries/Adafruit_SSD1306-esp8266-64x48/Adafruit_SSD1306.h](https://github.com/mcauser/Adafruit_SSD1306/blob/master/Adafruit_SSD1306.h)
